# 📚 Book Scraper

A web scraper built with Scrapy that automatically crawls 
all 50 pages of books.toscrape.com and extracts 1000+ book titles.

## Features
- Automatically follows pagination links
- Extracts all book titles
- Saves results to a text file

## Technologies
- Python
- Scrapy
- XPath

## How to Run
pip install scrapy
scrapy runspider book_crowel.py

## Output
book_title.txt containing 1000+ book titles
